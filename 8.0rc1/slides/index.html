Cylc Documentation§

Cylc (”silk”) is a workflow engine for cycling systems - it orchestrates distributed workflows of interdependent cycling tasks that may continue to run indefinitely.

Note

Cylc 8.0rc1 is a Cylc 8 release candidate. It is largely complete and can be used to manage real workflows, but it has not yet been heavily tested by users. Python 2 based Cylc 7 is stable and still available if needed for production systems.

Please report migration issues to the Cylc Discourse Forum (7-to-8 topic).
