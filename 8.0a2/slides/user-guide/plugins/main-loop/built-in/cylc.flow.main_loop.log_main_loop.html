Main loop plugin for monitoring main loop plugins.

Note

This plugin is for Cylc developers debugging main loop operations.

If matplotlib is installed this plugin will plot results as a PDF in the run directory when the suite is shut down (cleanly).

Coroutines

init Patch the timings for each plugin to use an unlimited deque.
