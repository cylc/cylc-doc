Cylc Documentation
==================

Cylc ("*silk*") is a workflow engine for cycling systems - it orchestrates
distributed suites of interdependent cycling tasks that may continue to run
indefinitely.

.. warning::

   Cylc 8 is beta software that is not yet recommended for production use.

   This documentation has not yet been fully updated and may contain errors.
